# Projeto 01 - Controle de Tarefas com Login

Tema: Lista de tarefas do dia a dia

## Login
O sistema utiliza login com CPF fixo para acesso.

CPF válido para teste:
12345678900

(O login aceita apenas este CPF para entrar no sistema)

## Funcionalidades
- Login com CPF
- Adicionar tarefas
- Remover tarefas
- Marcar como concluídas
- Contador de tarefas concluídas
- Mensagem quando não houver tarefas
- Logout (sair da conta)

## Tecnologias
- React Native
- Expo Snack
